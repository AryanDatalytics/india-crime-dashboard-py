# 🇮🇳 India Crime Data Analysis Dashboard

## 📌 Project Overview
A basic project that analyzes crime data across different cities in India.
The dashboard gives insights into:
- Crime distribution by age group
- Crime types
- City wise filtering
- Interactive visualizations

- Crime domain comparison
- Basic city based filtering
- Interactive dashboard

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the dashboard:
   streamlit run dashboard.py
